# Expense Tracker 💰

A simple Expense Tracker built using **HTML, CSS, and Vanilla JavaScript**.

## Features

### Phase 1
- Add Total Salary
- Add Expenses
- Calculate Remaining Balance
- Input Validation
- Dynamic Expense List

### Phase 2
- LocalStorage Support
- Delete Expense
- Chart.js Pie Chart

### Phase 3
- Download Expense Report (jsPDF)
- Threshold Warning (Balance below 10%)
- Currency Conversion (INR ↔ USD)

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage
- Chart.js
- jsPDF
- Frankfurter API

## How to Run

1. Clone the repository.
2. Open `index.html` in a browser.
3. Start tracking your expenses.
